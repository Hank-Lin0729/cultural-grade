## 🎯 我們的理念
文化是民族的靈魂，導覽是文化的橋樑。  
我們專注於培養專業的文化導覽師，讓每一位學員都能成為文化傳承的使者，  
用專業知識與熱忱態度，向世界展現台灣豐富多元的文化底蘊。
